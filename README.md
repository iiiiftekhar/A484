## Setup

1. **Clone the repository:**
   ```bash
   git clone https://github.com/iiiiftekhar/XQ7.git
   ```

2. **Navigate to the project directory:**
   ```bash
   cd XQ7
   ```

3. **Install dependencies:**
   ```bash
   npm install
   ```

4. **Run the development server:**
   ```bash
   npm start
   ```

5. **Open your browser and visit:**
   ```
   http://localhost:3000
   ```
